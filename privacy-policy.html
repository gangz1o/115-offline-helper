Privacy Policy

Effective date: February 12, 2026  |  Last updated: February 12, 2026

115 Offline Helper ("the Extension", "we", "us", or "our") is a browser extension that helps users detect magnet and ed2k links and push them to 115.com for offline downloading. This privacy policy explains how the Extension handles user data. We are committed to protecting your privacy and being transparent about our data practices.

1. Information We Collect

The Extension accesses the following types of data:

Data TypePurposeStored?
115.com authentication cookiesAuthenticate API requests to 115.com on your behalfLocally only (chrome.storage.local)
Clipboard contentDetect magnet/ed2k links when you copy text (opt-in only, requires user to enable "Auto detect links")Never stored; read in memory only and immediately discarded
User preferencesSave your settings (language, theme, save directory, auto-organize preferences)Locally only (chrome.storage.local)
Page content (link URLs)Scan the current page for magnet/ed2k links when auto-detect is enabledNever stored; processed in memory only

We do not collect personally identifiable information (PII) such as names, email addresses, phone numbers, or physical addresses.

2. How We Use Your Information

The data we access is used exclusively to provide the Extension's core functionality:

  • Authentication cookies — to communicate with 115.com APIs on your behalf (login status check, submit offline downloads, manage files)
  • Clipboard content — to detect and propose downloading magnet/ed2k links that you have copied. This feature is opt-in and disabled by default.
  • User preferences — to remember your settings between sessions
  • Page content — to identify and highlight magnet/ed2k links for one-click downloading. This feature is opt-in and disabled by default.

We do not use your data for advertising, analytics, marketing, profiling, or any purpose other than the core functionality described above.

3. Data Sharing and Disclosure

  • We do not sell, trade, or rent your data to anyone.
  • We do not transfer data to third-party servers, advertising platforms, data brokers, or information resellers.
  • We do not share data with any third party for any reason.
  • The only external communication is between your browser and 115.com servers (*.115.com), initiated solely to fulfill your download requests and manage your 115.com account.

4. Data Storage and Security

  • All persistent data (settings, cached cookies) is stored locally in your browser using chrome.storage.local. This data never leaves your device except for API requests to 115.com.
  • All network communication with 115.com uses HTTPS encryption.
  • Authentication cookies are only used for their intended purpose (115.com API authentication) and are never exposed to other services.
  • The Extension does not use any remote servers, databases, or cloud storage controlled by the developer.

5. Data Retention

  • User preferences are stored locally until you uninstall the Extension or manually clear the Extension's data.
  • Clipboard and page content are processed in real-time memory only and are never stored persistently. They are discarded immediately after link detection.
  • Authentication cookies are cached locally and cleared when you log out or uninstall the Extension.

Uninstalling the Extension removes all locally stored data.

6. User Rights

You have full control over your data and the Extension's behavior:

  • Access — You can view all stored settings in the Extension's Settings tab.
  • Modification — You can change any setting at any time through the Extension's interface.
  • Deletion — Uninstalling the Extension removes all stored data. You can also clear Extension data via Chrome's settings (chrome://extensions → Extension details → "Clear data").
  • Opt-out — The "Auto detect links" and clipboard monitoring features are disabled by default and require explicit opt-in. You can disable them at any time.
  • Permission revocation — You can revoke the optional <all_urls> permission at any time by disabling "Auto detect links" in the Extension settings.

7. Permissions Explained

PermissionWhy It Is Needed
storageSave your settings and cached authentication data locally in the browser
cookiesRead 115.com login cookies to authenticate API requests on your behalf
notificationsDisplay desktop notifications about download task completion or errors
activeTabAccess the current tab to detect magnet/ed2k links on the page you are viewing
scriptingInject the link detection content script into pages when "Auto detect links" is enabled
<all_urls> (optional)Only requested when you enable "Auto detect links". Allows the Extension to scan all web pages for magnet/ed2k links. You can revoke this permission at any time.

We follow the principle of least privilege: we only request the minimum permissions necessary for the Extension to function. The broadest permission (<all_urls>) is optional and requires explicit user consent.

8. Network Requests

The Extension communicates only with 115.com and its subdomains (*.115.com). Specifically:

Endpoint DomainPurpose
my.115.comCheck login status
qrcodeapi.115.comGenerate QR codes for login authentication
passportapi.115.comQR code login polling
webapi.115.comSubmit offline download tasks, manage files (move, rename, delete)
proapi.115.comAlternative API endpoints for file operations

No data is ever sent to any server other than those listed above. All requests use HTTPS.

9. Limited Use Disclosure

In compliance with the Chrome Web Store User Data Policy:

  • The Extension's use of information received from Chrome APIs adheres to the Chrome Web Store User Data Policy, including the Limited Use requirements.
  • Data obtained through the Extension is used solely to provide and improve the Extension's single purpose: detecting magnet/ed2k links and submitting them to 115.com for offline download.
  • We do not transfer user data to third parties except as necessary to provide the Extension's functionality (i.e., communication with 115.com).
  • We do not use or transfer user data for serving advertisements.
  • We do not use or transfer user data to determine creditworthiness or for lending purposes.
  • We do not sell user data.
  • Human access to user data does not occur.

10. Children's Privacy

The Extension is not directed to children under the age of 13. We do not knowingly collect personal information from children. If you are a parent or guardian and believe that your child has provided personal information through the Extension, please contact us so we can take appropriate action.

11. Changes to This Policy

We may update this privacy policy from time to time. Any changes will be reflected on this page with an updated "Last updated" date. We encourage you to review this page periodically. Continued use of the Extension after changes constitutes acceptance of the revised policy.

12. Contact Us

If you have any questions, concerns, or requests regarding this privacy policy or the Extension's data practices, please contact us through one of the following channels:

1. 我们收集的信息

本扩展访问以下类型的数据：

数据类型用途是否存储
115.com 认证 Cookie代表您向 115.com 发送 API 请求时进行身份验证仅本地存储（chrome.storage.local）
剪贴板内容检测您复制的文本中的 magnet/ed2k 链接（需用户主动开启"自动识别链接"后生效）从不存储；仅在内存中读取并立即丢弃
用户偏好设置保存语言、主题、保存目录、自动整理等设置仅本地存储（chrome.storage.local）

我们不会收集任何个人身份信息（PII），如姓名、电子邮件、电话号码或物理地址。

2. 信息的使用方式

我们访问的数据仅用于提供扩展的核心功能：

  • 认证 Cookie — 代表您与 115.com API 通信（检查登录状态、提交离线下载、管理文件）
  • 剪贴板内容 — 检测并提示下载您复制的 magnet/ed2k 链接。此功能默认关闭，需要用户主动开启。
  • 用户偏好设置 — 在不同会话之间记住您的设置
  • 页面内容 — 识别并高亮页面上的 magnet/ed2k 链接以支持一键下载。此功能默认关闭，需要用户主动开启。

我们不会将您的数据用于广告、数据分析、营销、用户画像或上述核心功能以外的任何目的。

3. 数据共享与披露

  • 我们不会向任何人出售、交易或出租您的数据。
  • 我们不会将数据传输至第三方服务器、广告平台、数据代理商或信息转售商。
  • 我们不会以任何理由与任何第三方共享数据。
  • 唯一的外部通信发生在您的浏览器和 115.com 服务器 (*.115.com) 之间，仅用于执行您的下载请求和管理您的 115.com 账户。

4. 数据存储与安全

  • 所有持久化数据（设置、缓存的 Cookie）均使用 chrome.storage.local 存储在您的浏览器本地。除了向 115.com 发送 API 请求外，数据不会离开您的设备。
  • 与 115.com 的所有网络通信均使用 HTTPS 加密。
  • 认证 Cookie 仅用于其预期目的（115.com API 认证），不会暴露给其他服务。
  • 本扩展不使用任何由开发者控制的远程服务器、数据库或云存储。

5. 数据保留

  • 用户偏好设置 — 在本地存储直到您卸载扩展或手动清除扩展数据。
  • 剪贴板和页面内容 — 仅在实时内存中处理，从不持久化存储。链接检测完成后立即丢弃。
  • 认证 Cookie — 在本地缓存，在您注销登录或卸载扩展时清除。

卸载本扩展将删除所有本地存储的数据。

6. 用户权利

您可以完全控制您的数据和扩展的行为：

  • 查看 — 您可以在扩展的"设置"选项卡中查看所有存储的设置。
  • 修改 — 您可以随时通过扩展界面更改任何设置。
  • 删除 — 卸载扩展将删除所有存储的数据。您也可以通过 Chrome 设置清除扩展数据（chrome://extensions → 扩展详情 → "清除数据"）。
  • 退出 — "自动识别链接"和剪贴板监控功能默认关闭，需要用户明确开启。您可以随时关闭它们。
  • 权限撤销 — 您可以随时在扩展设置中关闭"自动识别链接"来撤销可选的 <all_urls> 权限。
